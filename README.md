# PEG Security Website

Professional security services website for PEG Security - South Africa's premier protection force.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase (Database & Storage)
- Framer Motion (Animations)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env.local
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on Vercel: https://peg-security-nextjs.vercel.app/
